Web Corporativa LaTabla

LaTabla es una página web corporativa desarrollada para un negocio real especializado en embutidos y productos gastronómicos. El objetivo del proyecto fue mejorar la presencia digital del establecimiento y ofrecer a los clientes una forma sencilla de conocer sus productos, ubicación e información de contacto.

La web ha sido desarrollada utilizando HTML y CSS, priorizando la simplicidad, la velocidad de carga y una experiencia de usuario clara y atractiva. Todo el contenido ha sido estructurado para transmitir la identidad del negocio y facilitar el acceso a la información más relevante.

Funcionalidades principales

  • Página principal con información del negocio.
  • Historia y presentación del establecimiento.
  • Mapa interactivo con ubicación real.
  • Productos destacados.
  • Carta completa del local.
  • Galería de imágenes de productos.
  • Botón de contacto directo mediante WhatsApp.
  • Diseño responsive para móviles y tablets.
  • Páginas legales y políticas de privacidad.

Tecnologías utilizadas

HTML, CSS, Responsive Design, Diseño Web, UI/UX y optimización de experiencia de usuario.

Objetivos del proyecto

El principal objetivo fue crear una web atractiva, funcional y alineada con la imagen del negocio, mejorando su visibilidad online y facilitando el acceso de clientes a la información, productos y servicios ofrecidos por el establecimiento.

Aprendizajes obtenidos

Este proyecto me permitió reforzar conocimientos de desarrollo Front-End, diseño web responsive y experiencia de usuario, así como trabajar sobre las necesidades reales de un negocio para transformarlas en una solución digital funcional.
