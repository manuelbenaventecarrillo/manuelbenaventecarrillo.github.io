Proyectos


CasaCuidadoPro (TFG) | Sistema de Gestión de Cuidadores

Aplicación desarrollada como Trabajo de Fin de Grado, orientada a la gestión integral de cuidadores, usuarios y servicios de asistencia domiciliaria. El sistema permite organizar turnos, asignar cuidadores, administrar perfiles de usuarios y centralizar toda la información necesaria para una gestión eficiente y estructurada. El proyecto incluye diseño de base de datos relacional, lógica de backend y una interfaz funcional pensada para facilitar su uso en entornos reales. Ha sido desarrollado aplicando buenas prácticas de programación, estructuración de código y optimización de procesos, con el objetivo de ofrecer una solución escalable y útil en el ámbito de la atención y el cuidado de personas.
