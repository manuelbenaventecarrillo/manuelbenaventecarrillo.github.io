Almacén Digital LaTabla

El Almacén Digital de LaTabla es un proyecto personal desarrollado por iniciativa propia con el objetivo de digitalizar y optimizar la gestión interna del negocio familiar "LaTabla". Surge de la necesidad de sustituir procesos manuales por una solución tecnológica capaz de centralizar información y mejorar la eficiencia operativa del establecimiento.

La aplicación está siendo diseñada para gestionar inventario, productos, stock, movimientos de almacén y ventas mediante una interfaz intuitiva y accesible. El objetivo es disponer de una herramienta que permita consultar información, registrar movimientos y facilitar la toma de decisiones de forma rápida y eficiente.

Funcionalidades principales

  • Control de inventario y stock en tiempo real.
  • Registro de entradas y salidas de productos.
  • Gestión de productos, categorías y proveedores.
  • Control de ventas y movimientos del almacén.
  • Base de datos centralizada.
  • Consultas rápidas de stock y productos.
  • Panel de administración del sistema.
  • Historial de movimientos y trazabilidad.
  • Análisis de datos mediante Inteligencia Artificial.
  • Chat con Inteligencia Artificial integrado.

Tecnologías utilizadas

HTML, CSS, JavaScript, Node.js, SQL, bases de datos relacionales (PostgreSQL / MySQL), JSON, diseño UI/UX, arquitectura de aplicaciones web, control de versiones con Git y GitHub, uso de Lovable como herramienta de desarrollo asistido, e integración de Inteligencia Artificial para análisis de datos, gestión de inventario y chat inteligente de asistencia.

Motivación del proyecto

A diferencia de otros proyectos personales, este desarrollo nace de una necesidad real detectada dentro de un entorno de trabajo existente. El objetivo es aplicar la tecnología para resolver problemas cotidianos y optimizar procesos que actualmente consumen tiempo y recursos de forma innecesaria, mejorando la eficiencia operativa del negocio.

Aprendizajes y objetivos

Este proyecto me está permitiendo profundizar en desarrollo Full Stack, diseño de interfaces, bases de datos, automatización de procesos e integración de Inteligencia Artificial. Además, representa una oportunidad para construir una herramienta con impacto real, aplicando conocimientos técnicos en un entorno práctico mientras continúo ampliando mis competencias como desarrollador.
